Research and Creation Fellowship 2017


Title: MEDIATED REALITY: REVISED

Call Type: Fellowship

Award: USD$10,000 + CAC_LAB's facilities and equipment.

Starting Date: March 20th 2017

Application Deadline: All applications must be submitted electronically by 23:59 (Beijing Time), Friday, January 20th , 2017.

Please answer and complete the following application form. Take in account the following guidelines:

1. All fields are marked with * are required.

2. If the applicant doesn't have an online portfolio website, he/she must provide a list of URLs using the "Other Links" field in this form. We recommend to use vimeo for video samples, flickr and soundcloud for audio samples.

3. If the applicant does have an online portfolio and still wants to provide extra URLs for additional work samples he/she is free to do so by using the "Other Links" field in this form.

4. The Project Proposal PDF should be uploaded as a clear document containing at least the following items:

  • Genral Description of the Project
  • Conceptual Framework
  • Technical Description (software / hardware)
  • Tell us how does your project makes use of our realtime MOCAP (optitrack) system.
  • Detailed Time Schedule and Development Plan
  • Budget: How do you plan to use the funds provided by the fellowship (materials, prototyping, etc.)? Do you have extra funding? if so please mention on your document.
    VERY IMPORTANT: As part of the fellowship's process, CAC will host a show for the fellows in our gallery space. Any cost for installation and/or shipping has to be considered as part of your fellowship's budget, so please include it in your budget section.

5. Finally, please upload the required files (maximum size 3.5 Mb): Proposal PDF and your CV/Resume PDF.

IMPORTANT NOTE: Please fill out this form when you have all the information ready. The system DOES NOT save incomplete submissions.
